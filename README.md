# Planet Explorer (Ionic Pet-Project)

Цей проект є навчальним веб-додатком, розробленим для демонстрації роботи з Ionic Framework. Проект реалізований як односторінковий додаток (SPA), що дозволяє переглядати інформацію про планети Сонячної системи.

## Про проект
Метою створення цього проекту було вивчення можливостей сучасних веб-технологій та компонентів Ionic, зокрема:
- Робота з Custom Elements для створення модульних сторінок.
- Реалізація клієнтської маршрутизації (Router) для навігації між головною сторінкою та сторінкою деталей.
- Візуалізація даних у вигляді карток (Cards), акордеонів (Accordions) та чіпів (Chips).

## Функціональні можливості
- Перегляд списку планет з коротким описом.
- Детальна сторінка з фізичними характеристиками (маса, температура, склад атмосфери).
- Інтерактивний інтерфейс для швидкого пошуку інформації.

## Технології
- HTML5, JavaScript (ES6+).
- Ionic Framework (Core).
- CSS (для адаптивного дизайну).
